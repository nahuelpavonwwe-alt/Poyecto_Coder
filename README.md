# 🛒 Simulador de Ecommerce de prendas basicas.

## Descripción

Este proyecto es un simulador de ecommerce desarrollado como práctica de
JavaScript.\
Permite visualizar productos, seleccionar variantes como color y talle,
agregarlos a un carrito y simular una compra.

El objetivo del proyecto es aplicar conceptos fundamentales de
desarrollo web como manipulación del DOM, manejo de eventos, uso de JSON
y almacenamiento en localStorage.

------------------------------------------------------------------------

## Funcionalidades

-   Visualización dinámica de productos
-   Carga de productos desde archivo JSON
-   Selección de color y talle
-   Agregar productos al carrito
-   Eliminación de productos del carrito
-   Persistencia del carrito con localStorage

------------------------------------------------------------------------

## 🧰 Tecnologías utilizadas

-   HTML5
-   CSS3
-   JavaScript (ES6)
-   JSON
-   Librerias (SweetAlert2)

------------------------------------------------------------------------

## 🚀 Cómo ejecutar el proyecto

1.  Clonar o descargar el repositorio
2.  Abrir la carpeta del proyecto
3.  Ejecutar el archivo `index.html` en el navegador
